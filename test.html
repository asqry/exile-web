Command Usage Description
.help .help Displays the help/command list
.help config .help [config, cf] Displays the configuration commands
.help moderation .help [moderation, mod] Displays the moderation commands
.help economy .help [economy, eco] Displays the economy commands
.help music .help music Displays the music commands
.help other .help [other, misc] Displays the miscellaneous commands
.serverinfo .serverinfo Displays information about the current server
.botinfo .botinfo Displays information about the bot
.new .new [issue] Creates a new ticket to solve all of your problems
.close .close Closes the current ticket, only works within ticket channels
.suggest .suggest [suggestion] Sends your suggestion in the set suggestion channel
Command Usage Description
.joinchannel .joinchannel [channelID] Sets the channel for join messages
.announcechannel .announcechannel [channelID] Sets the channel in which announcements will be sent
.modlogchannel .modlogchannel [channelID] Sets the channel that all moderation commands will be logged in
.logchannel .logchannel [channelID] Sets the channel for logging all other commands
.suggestchannel .suggestchannel [channelID] Sets the channel in which suggestions will be sent
.musicchannel .musicchannel [voiceChannelID] Sets the channel in which the bot will join when .play is executed
.autorole .autorole [roleName, roleID] Sets the role that will be automatically given to new users
Command Usage Description
.announce .announce [announcement] *Tags everyone Sends an announcement in the specified announcement channel, tags everyone
.clear .clear [# of messages] Clears the specified amount of messages from the channel
.say .say [message] The bot outputs the given message
.warn .warn [@user] [reason] Warns the specified user
.kick .kick [@user] [reason] Kicks the specified user from the server
.ban .ban [@user] [reason] Bans the specified user from the server
.mute .mute [@user] [time] [reason] Mutes the user for the specified amount of time
.addrole .addrole [@user] [roleName, roleID] Gives the user the specified role
.removerole .removerole [@user] [roleName, roleID] Removes the specified role from the user
Command Usage Description
.coins .coins Checks your coin balance, cannot check other's balance
.level .level Checks your level and remaining XP, cannot check other's level
Command Usage Description
.play .play [youtube link] Plays the specified link in the dedicated music channel
.leave .leave Forces the bot out of the music channel
Command Usage Description
.8ball .8ball [question] Let the 8 ball make your decision
