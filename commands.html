Exile BOT

Exile is a modern and versatile bot with a bunch of great commands to fit your needs. With a premium feature coming soon featuring custom commands made personally, by Exile's Developers. Below is a list of Exile's many commands.

Command Usage Description
!help !help Displays the help-related commands
!help all !help [all, a] Displays a list of all commands
!help moderation !help [moderation, mod] Shows moderation commands (Moderator only marked with *)
!help fun !help [fun, games] Displays all game commands
!report !report [@user] [reason] Logs a report against a certain users
* !ban !ban [@user] [reason] Bans a certain user from the server
* !kick !kick [@user] [reason] Kicks a certain user from the server
* !addrole !addrole [@user] [roleName, roleID] Gives the user a certain role DO NOT @ THE ROLE
* !removerole !removerole [@user] [roleName, roleID] Takes a certain role away from the user DO NOT @ THE ROLE
